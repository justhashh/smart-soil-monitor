# 🌱 Smart Soil Moisture & Temperature Monitoring System

## 📌 Overview

This Arduino project monitors soil moisture and temperature conditions in real time using sensors and LED alerts.

---

# ⚙️ Components Used

* Arduino UNO
* Soil Moisture Sensor
* Temperature Sensor
* LED
* Resistor
* Breadboard
* Jumper Wires

---

# 🔧 Features

* Soil moisture monitoring
* Temperature monitoring
* Dry soil LED alert
* Rapid temperature rise detection
* Serial Monitor status messages

---

# 💡 Working

* Reads soil moisture and temperature values
* Detects dry soil conditions
* Detects sudden temperature increases
* Displays warnings through Serial Monitor and LED

---

# ▶️ Output Alerts

* `"SOIL IS DRY - WATER NEEDED"`
* `"RAPID TEMP INCREASE!"`
* `"POSSIBILITY OF PEST ATTACK!"`
